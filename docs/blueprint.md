# **App Name**: Motel Manager Pro

## Core Features:

- Real-time Dashboard: Displays key performance indicators (KPIs) such as occupied rooms, turno incomes, and net profits.
- Automated Turn Calculation: Automatically calculates the work shift based on the 'America/Mexico_City' timezone, including handling shifts that cross midnight and assigning a operational date based on the shift time.
- Room Status Management: Dynamically updates room statuses (Available, Occupied, Cleaning, Maintenance) with visual indicators based on timers and occupancy duration.
- Integrated Checkout: Closes consumptions, calculates stay duration, check controls inventory and updates data, handles all transaction
- Vehicle Report Integration: Cross-validates plates from checkin input form against a list of blacklisted vehicles to ensure compliance.
- Generative AI Consultant: AI-powered chat tool connected to Google Gemini that interprets data and responds to queries such as profit calculations and discrepancy identification using internal metrics from the transactions tables.
- User Role Security: Implements password-protected modals for sensitive operations such as deleting expenses or modifying system settings to provide an auditable user experience.

## Style Guidelines:

- Primary color: Rose (#E11D48) for the brand's main identity, creating a vibrant feel.
- Background color: Slate (#F8FAFC) for a neutral base, ensuring the vibrant colors stand out.
- Accent color: Violet (#8B5CF6) for interactive elements, offering a high-contrast complement to the primary rose color.
- Headline font: 'Poppins', a geometric sans-serif for short text, creating a precise and contemporary feel.
- Body font: 'PT Sans', a humanist sans-serif to complement Poppins that ensures readability.
- Code font: 'Source Code Pro' for displaying code snippets.
- Lucide-react icons to maintain a consistent and modern aesthetic throughout the interface.
- Rounded corners (rounded-2xl, rounded-xl) and soft shadows (shadow-lg) to create a friendly and modern user interface.
- Subtle transitions and animations on state changes to enhance user experience.
